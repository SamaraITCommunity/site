По запросам сообщества запускаем SITC Game Jam! Неожидано для нас самих - без партнёров, без спонсоров, без судей - только участники, которые сами выберут победителя.

Что такое Game Jam

Есть тема и несколько критериев которым должен соответсвовать проект. Вы выбираете технологии, фреймворки и команду или делаете всё в одиночку. Разрабатываете и отсылаете проект на страницу джэма, а затем голосуете за понравившиеся проекты.

Как прошёл первый Game Jam?

В первом Game Jam участвовал только Никита Пузанков (@humb1t) с проектом Jack O Dubstep - надеемся на более активное участие в будущем.
