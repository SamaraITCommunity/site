Книжная Барахолка

Чтобы воспользоваться книжной барахолкой используйте таблицу. Для добавления вашей книги в барахолку - используйте ссылку на комментирование документа и укажите данные для заполнения нашим администратором. Для получения книги - напишите её владельцу, а затем добавьте комментарий о перемещении книги, с указанием ваших данных для связи.
